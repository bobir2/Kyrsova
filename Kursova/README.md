# Kursova — Інформаційна система складу продуктів харчування

Десктопний застосунок для обліку продуктів харчування на складі.

## Стек технологій

- **C#** (.NET 8)
- **WPF** — графічний інтерфейс
- **SQLite** — локальна база даних
- **MVVM** — архітектура застосунку

## Можливості

- **Панель керування** — статистика, низькі залишки, терміни придатності
- **Продукти** — додавання, редагування, пошук, фільтр за категорією
- **Категорії** — управління групами товарів
- **Постачальники** — контактні дані постачальників
- **Рух товарів** — надходження та відвантаження з автоматичним оновленням залишків

SQLite-файл створюється автоматично при першому запуску:

`%LOCALAPPDATA%\FoodWarehouse\warehouse.db`

## Структура проєкту

```
Kursova/
├── Data/           — ініціалізація БД та репозиторій
├── Models/         — моделі даних
├── ViewModels/     — логіка інтерфейсу (MVVM)
├── Views/          — XAML-екрани та діалоги
└── Converters/     — конвертери для прив'язки даних
```
